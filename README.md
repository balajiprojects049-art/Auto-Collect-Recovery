# Auto Collect Recovery Services

A modern, responsive vehicle recovery website built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **Animations**: Smooth page transitions and interactions using Framer Motion.
- **Routing**: Client-side routing with React Router Toolkit.
- **Components**: Reusable UI components for consistent design.
- **Forms**: Working contact and quote forms (frontend validation).

## Project Structure

- `src/components`: Reusable UI components (Header, Footer, ServiceCard, Layout).
- `src/pages`: Page components (Home, About, Services, Contact, Quote).
- `src/data`: Static data files (services.json).
- `src/assets`: Images and icons.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Deployment

This project is ready for deployment on Vercel or Netlify.
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

## Technology Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
